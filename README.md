# cyber-edit
Text editor for the Cyber desktop environment.

# Dependencies
<code>https://github.com/cyberos/meuikit</code>

# Building
<code>mkdir build</code>
<code>cd build</code>
<code>cmake ..</code>
<code>make</code>

# Installing
If you are in the build folder, go back:
<code>cd ..</code>
Then, run install.sh:
<code>./install.sh</code>
